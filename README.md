# Data Science & Analytics Internship Tasks

## Overview

This repository contains my completed tasks for the Data Science & Analytics Internship at DevelopersHub Corporation.

The projects demonstrate practical applications of Data Analysis, Data Visualization,
Machine Learning Classification, and Regression using Python and popular data science libraries.

## Tasks Completed

### Task 1: Iris Dataset Analysis

#### Objective
Explore and visualize the Iris Dataset to understand feature relationships and data distributions.

#### Key Activities
- Loaded and explored the Iris Dataset.
- Analyzed dataset structure and summary statistics.
- Created Scatter Plots.
- Created Histograms.
- Created Box Plots.
- Performed Exploratory Data Analysis (EDA).

#### Skills Demonstrated
- Pandas Data Analysis
- Data Visualization
- Exploratory Data Analysis (EDA)
- Matplotlib & Seaborn

### Task 4: Insurance Charges Prediction

#### Objective
Predict medical insurance charges based on customer attributes such as age, BMI, smoking status, and region.

#### Key Activities
- Data Cleaning and Preprocessing
- Exploratory Data Analysis
- Feature Relationship Analysis
- Linear Regression Model Training
- Model Evaluation using:
  - MAE (Mean Absolute Error)
  - RMSE (Root Mean Squared Error)

#### Skills Demonstrated
- Regression Modeling
- Feature Analysis
- Data Visualization
- Model Evaluation

### Task 5: Personal Loan Acceptance Prediction

#### Objective
Predict whether a customer is likely to accept a personal loan offer using demographic and financial information.

#### Key Activities
- Data Exploration and Cleaning
- Label Encoding of Categorical Variables
- Exploratory Data Analysis
- Logistic Regression Model Training
- Model Evaluation using:
  - Accuracy Score
  - Classification Report
  - Confusion Matrix

#### Model Performance

**Accuracy Score:** 88.51%

#### Skills Demonstrated
- Classification Modeling
- Feature Engineering
- Machine Learning
- Business Insight Generation
