# Iris Flower Classification

## Objective

The objective of this project is to analyze the Iris flower dataset and build a machine learning model to classify flower species based on their features.

## Dataset

The Iris dataset contains 150 samples with the following features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width
* Species

## Approach

1. Data Cleaning and Preparation
2. Exploratory Data Analysis (EDA)
3. Data Visualization
4. Train-Test Split
5. Random Forest Classification
6. Model Evaluation

## Results and Insights

* The dataset was clean with minimal preprocessing required.
* Petal Length and Petal Width were the most important features.
* A Random Forest Classifier was trained on the dataset.
* The model achieved 100% accuracy on the test set.
* The confusion matrix showed no misclassifications.
